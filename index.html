GA4 vs Shopify Data Verification

Shopify Admin data cross-referenced with GA4 — Jan 19 – Feb 15, 2026

Why two data sources? GA4 tracks user behavior (browsing, clicks, events) while Shopify is the source of truth for transactions. Cubelelo uses GoKwik for 98.5% of checkout, which bypasses Shopify’s native checkout — causing GA4 to undercount purchases and revenue.

Key Finding: Eurus Supports Custom Metafield Badges

This means difficulty level badges, speed/casual badges, and puzzle type indicators can all be implemented on Eurus without any theme switch. Source: Eurus Metafield Badge Documentation
